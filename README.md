# ManuOS v1.0 - Terminal OS for OpenComputers

Professional terminal OS for Minecraft OpenComputers.

## Features
- Linux-like filesystem
- Full shell with 14+ commands
- 3 built-in apps
- Configuration system

## Quick Start
`lua /init.lua`

## Commands
pwd, cd, ls, cat, mkdir, rm, touch, time, mem, help, lua, exit

## Apps
- Calculator: usr/bin/calculator.lua
- Editor: usr/bin/editor.lua
- Files: usr/bin/files.lua
